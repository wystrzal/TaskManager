## Setup
To run API open TaskManager.API.sln with Visual Studio,
change from IIS Express to TaskManager.API and press F5.

To run SPA open TaskManager folder with Visual Studio Code and
`cd TaskManager-SPA`
`npm install`
`ng serve`
Open http://localhost:4200
